Responses APIに対応。最新のOpenAIモデル（GPT-5.2系）でレコード内容から回答を生成します。

推奨
gpt-5.2
最新・標準。高品質なテキスト生成に最適。
gpt-5.2-pro

OpenAI設定

組織のセキュリティポリシーに沿って管理してください。
Responses API対応モデルを選択してください（OpenAI OpenAPI spec v2.3.0より）。
`reasoning_effort`。gpt-5.1系で利用可能なThinking強度を指定します。
AIの役割やトーンを指定します。未入力時は汎用アシスタントとして動作します。

kintoneフィールド設定

AIに渡す本文のフィールドコード。
AIの回答を保存するフィールドコード。
